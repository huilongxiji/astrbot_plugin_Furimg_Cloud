## ✨ v1.0.0 内容

- **用户核心功能**：本插件有着 随机兽图、来只（综合查询）、来只毛，来只兽 等等获取furry毛图或插画的方式。
- **账户系统（基础）**：当前 1.0.0 版本有着基础的账户系统，管理员可配置自己的账户来观看特殊格式的图片，比如私密图片（仅登录账号所有的图片），本版本写了登录，自动登录，以及token的恒心指令。
- **设置相关**：插件可填写自己的账户以及审核群聊（以后版本更新实装投稿之后可用），使用者可自行选择是否开启自动登录（不想看到插件启动的时候因未填写账户而导致的登陆失败警告的时候可用）设置里面的图片像款区域属于下一部分更新内容，暂时不可用。

## 🔧 插件功能列表（当前版本）

| 指令 | 参数 | 权限 | 预计返回内容 | 功能介绍 |
| --- | --- | --- | --- | --- |
| /随机兽图 | 无 | 所有人 | 图文（富媒体） | 随机获取一张兽图或毛毛照片或插画 |
| /来只 | 数字ID或者名称（关键字） | 所有人 | 图文（富媒体） | 获取一张指定的兽图或毛毛照片或插画 |
| /来只毛 | 若携带名称则为关键字查询，不带参数则随机 | 所有人 | 图文（富媒体） | 随机或者指定获取一张毛毛照片 |
| /来只兽 | 若携带名称则为关键字查询，不带参数则随机 | 所有人 | 图文（富媒体） | 随机或者指定获取一张兽兽插画 |
| /兽云验证码 | 无 | 超管 | 纯图片 | 获取一张图形验证码 |
| /兽云登录 | 无 | 超管 | 文本或图片 | 使用token进行登录操作 |
| /兽云更新登录令牌 | 无 | 超管 | 纯文字 | 更新账户唯一登录令牌 |
